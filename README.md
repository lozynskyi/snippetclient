# snippetclient

## Build Setup

```bash
# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev

# add deploy to surge and scaleway
$ npm run generate

# build for production and launch server
$ npm run build
$ npm run start

# generate static project
$ npm run generate
```
